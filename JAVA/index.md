## Controller

> 控制类，处理 HTTP 请求，返回 HTTP 相应

* `@RestController`
    这个控制器返回数据
* `@GetMapping`
    当有 GET 请求发送到特定的 URL 时，应该调用这个方法来处理请求
* `@RequestParam`
    从请求中获取 key 的 value，并且赋值给某个变量
