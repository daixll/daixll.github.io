# 算法竞赛是什么？

本质上，就是做题，而且很像做数学题。

只不过，算法竞赛的题目，会抽象一层，让你去思考，如何用计算机去解决这个问题。

具体的，就是给你一些输入，让你输出一些东西。

# 为什么要参加算法竞赛？

1. 至少付出一年的时间
2. 付出不一定得到回报
1. 不能免试专升本
2. 不能推免研究生
