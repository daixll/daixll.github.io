# 实践与认知

> 人不能两次踏入同一条河流

1. 所有东西时时刻刻都是在发生变化的，变化是常态
2. 珍惜当下
3. 理解不可逆性
4. 灵活应对生活中的一切
5. 寻求平衡，例如生活和工作的平衡

> 人甚至一次都不能踏入同一条河流

河流的水在不断的流动，河道的构成元素也在变化，从这个角度上来说，河流也在不断的变化

物体是绝对的运动，相对的静止

> 庄子，生即是死，死既是生

忽略了相对的静止，人生是有阶段性的

> 时间和空间

时间标志物质运动过程的持续性

空间指运动着的物质的广延性

没有脱离空间的时空

> 观念的东西不外是移入人的头脑并在人的头脑中改造过的物质的东西而已

认知水平不一样，看到的东西就不一样

> 意识的能动性

1. 意识活动的目的性和计划性

2. 意识活动的能动性和创造性

3. 意识具有变革客观世界的作用

4. 意识具有调控人的行为活动和生理活动的作用
