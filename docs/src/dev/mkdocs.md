### 我的网站

1. 安装 `python`

    ```shell
    sudo apt install python3 python3-pip
    ```

2. 安装 `mkdocs`

    ```shell
    pip install mkdocs
    ```

3. 安装 `mkdocs-material`

    ```shell
    pip install mkdocs-material
    ```

4. 拉取网站

5. 生成网站

    ```shell
    python -m mkdocs build
    ```
